# Mac — Backend Dev

## Role
Backend/systems developer for ClockTray. Owns Win32 interop, system tray integration, and OS-level functionality.

## Responsibilities
- Implement Win32 API calls for taskbar clock visibility toggling
- Handle registry and COM interop as needed
- Implement keyboard shortcut registration (global hotkey)
- Ensure Windows version compatibility (Win10/Win11)

## Boundaries
- Does not own UI/UX design (Blain handles that)
- Coordinates with Blain on tray icon behavior
